## 9 — Add pagination to ALL list endpoints

**Files:** All route files with GET list endpoints

**Problem:** Every list endpoint returns ALL records with no limit.

**Fix:** Create a shared pagination helper:
```typescript
// src/lib/pagination.ts
export function parsePagination(query: any) {
  const MAX_LIMIT = 200;
  const limit = Math.min(parseInt(query.limit as string) || 50, MAX_LIMIT);
  const offset = Math.max(parseInt(query.offset as string) || 0, 0);
  return { limit, offset };
}
```

Apply to: boards, votes, meetings, minutes, documents, tasks, people, audit endpoints:

## 13 — Add `rel="noopener noreferrer"` to sanitized links

**File:** `artifacts/api-server/src/lib/sanitize.ts`

**Problem:** `sanitizeRichHtml` allows `<a href>` but doesn't enforce safe attributes.

**Fix:**
```typescript
export function sanitizeRichHtml(input: string): string {
  return sanitizeHtml(input, {
    allowedTags: ["b", "i", "u", "strong", "em", "p", "br", "ul", "ol", "li",
                  "h1", "h2", "h3", "h4", "a", "blockquote", "table", "thead",
                  "tbody", "tr", "th", "td"],
    allowedAttributes: {
      a: ["href", "target"],
    },
    transformTags: {
      a: sanitizeHtml.simpleTransform("a", { rel: "noopener noreferrer", target: "_blank" }),
    },
    disallowedTagsMode: "discard",
  });
}
```

## 16 — Fix task number race condition

**File:** `artifacts/api-server/src/routes/tasks.ts`

**Problem:** `getNextTaskNumber()` reads count then inserts — concurrent requests get duplicate numbers.

**Fix:** Use a DB sequence or wrap in a transaction with a lock:
```typescript
async function getNextTaskNumber(): Promise<string> {
  const year = new Date().getFullYear();
  const [result] = await db.execute(sql`
    SELECT COUNT(*) as count FROM ${tasksTable}
    WHERE created_at >= ${new Date(year, 0, 1).toISOString()}
    FOR UPDATE
  `);
  const seq = (Number(result.count) + 1).toString().padStart(3, "0");
  return `TASK-${year}-${seq}`;
}
```
